#### 登陆注册

baseURL:

方法：post

传输：json模式

输出：

| 参数名   | 参数格式 | 备注     |
| -------- | -------- | -------- |
| username | string   | 用户名   |
| userpass | string   | 用户密码 |

返回：

| 参数名  | 参数格式 | 备注                                |
| ------- | -------- | ----------------------------------- |
| errcode | int      | 0/404（0为成功，404为用户名被占用） |

#### 弹幕

baseURL:

方法：post

传输：json模式

输出:

| 参数名   | 参数格式 | 备注     |
| -------- | -------- | -------- |
| username | string   | 用户名   |
| content  | string   | 弹幕内容 |
| time     | string   | 发送时间 |

返回：

| 参数名  | 参数格式 | 备注                            |
| ------- | -------- | ------------------------------- |
| errcode | int      | 0/404（0为成功，404为发送失败） |
